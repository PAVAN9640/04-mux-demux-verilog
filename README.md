# MUX and DEMUX using Verilog

Digital Design Project #4: Implementation and simulation of 2:1 Multiplexer and 1:2 Demultiplexer using Verilog HDL.

## Overview

This project demonstrates the design and simulation of fundamental data routing circuits used in digital systems.

## Modules Implemented

- 2:1 Multiplexer
- 1:2 Demultiplexer

## Tools Used

- Verilog HDL
- Xilinx Vivado

## Concepts Learned

- Data Selection
- Data Routing
- Combinational Logic Design
- Verilog HDL
- Functional Simulation

## Conclusion

Successfully implemented and verified MUX and DEMUX circuits using Verilog HDL.
